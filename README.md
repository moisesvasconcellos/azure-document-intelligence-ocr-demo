# 📄 Azure Document Intelligence - OCR Demo

## 📌 Objetivo
Demonstrar o uso do Azure Document Intelligence para extração de texto (OCR) a partir de imagens e documentos.

---

## 🧠 Tecnologias utilizadas
- Azure Document Intelligence
- OCR (Read Model)
- JSON Output

---

## 📂 Estrutura do projeto
- inputs/ → imagens utilizadas
- outputs/ → resultado extraído pela IA

---

## 🚀 Como funciona

1. Upload da imagem no Azure  
2. Execução do modelo OCR (Read)  
3. Extração automática de texto  
4. Retorno estruturado em JSON  

---

## 📊 Resultado

A IA foi capaz de:
- Identificar texto com alta precisão  
- Separar por linhas e parágrafos  
- Estruturar o conteúdo  

---

## 💡 Aplicações reais

- Leitura de notas fiscais  
- Automação de processos  
- Digitalização de documentos  
- Extração de dados para sistemas  

---

## 🔐 Arquitetura

App → Backend → Azure AI → JSON → Sistema
